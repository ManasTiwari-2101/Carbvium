# Carbvium - Requirements and setup notes

This file lists the runtime and project requirements to run the Carbvium application (backend + frontend).

System/runtime requirements
- Node.js: >= 18.14.0 (LTS recommended)
- npm: >= 9 (or `yarn` if preferred)
- Docker & Docker Compose: optional but recommended for containerized setup
- PostgreSQL: 14+ (required if you run a local/Postgres-backed instance; Supabase can be used instead)

Environment variables (examples)
- Backend: create a `.env` in `Carbvium_Backend/` with at least:
  - PORT=3000
  - DATABASE_URL=postgres://user:pass@host:port/dbname   # or Supabase connection URL
  - SUPABASE_URL=...
  - SUPABASE_KEY=...

Install & run (basic)
1) Install backend dependencies
   cd Carbvium_Backend
   npm install

2) Install frontend dependencies
   cd ../Carbvium_Frontend
   npm install

3) Run backend in development
   cd Carbvium_Backend
   npm run dev

4) Run frontend in development
   cd Carbvium_Frontend
   npm run dev

Notes
- If you want to run the whole project with Docker, the repository includes `docker-compose.yml` at the root. Adjust environment variables and Postgres credentials before `docker-compose up`.
- The backend expects either a local Postgres instance or Supabase; set `DATABASE_URL` or `SUPABASE_*` accordingly.
